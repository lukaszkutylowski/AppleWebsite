# **AppleWebsite**

This is a web application which give possibility add comment and read all comments with using ReactJS client, Spring Boot server, MySQL database.

## Used technologies:
* Spring MVC
* Spring Boot
* JPA
* Repository
* OOP
* HTML
* CSS
* JavaScript
* ReactJS
* MySQL
* IntelliJ
* Visual Studio Code
